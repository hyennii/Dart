## Dart#1 기본기

### double
: 실수
```Dart
    double number1 = 2.5;
```

### boolean
: 맞다/틀리다
```Dart
    bool isTrue = true;         // bool 이라고 씀
```

### String
: 글자타입
```Dart
    String name = '혜원';        // 맨 앞 대문자 써야함
```

### var와 String의 차이점
: var는 type을 오른쪽 값으로 인해 유추
```Dart
    var name3 = '혜원';     // name3 는 string이 되고, 
    var number = 20;        // number 는 int가 됨
```
